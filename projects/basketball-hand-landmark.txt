----------------------------------------------------------------------
QUICK “GOOD FEEL” TUNING CHEAT SHEET
----------------------------------------------------------------------
If shots feel too weak:
- Increase forwardSpeed multiplier (12 -> 14/15)
- Increase Y impulse multiplier (8.5 -> 9.5)

If shots feel too wild:
- Reduce est.vx / est.vy influence in shootBall()
- Increase hand anchor smoothing (lerp 0.35 -> 0.45)

If grabs are hard:
- Increase nearBall threshold (1.25 -> 1.5)
- Relax pinchOn (0.43 -> 0.46)

If accidental grabs happen:
- Tighten pinchOn (0.43 -> 0.40)
- Lower nearBall threshold (1.25 -> 1.1)
